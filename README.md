# React Basic - Todo App

Aplicación básica de tareas con React.

## Tecnologías

- React
- Custom Hooks
- localStorage

## Instalación

```bash
npm install
npm start
```

## Hooks personalizados

- `useLocalStorage` - Persistencia en localStorage
- `useTodos` - Lógica de tareas

## Propósito

Proyecto de práctica para aprender React y custom hooks.
